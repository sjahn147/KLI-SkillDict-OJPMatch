# 한국형 숙련사전 구축 및 구인공고 매칭 프로젝트

## 프로젝트 개요
- 목적: 텍스트 분석을 통한 한국형 숙련사전 구축 및 온라인 구인공고 자동 매칭 시스템 개발
- 기간: 2023년 1월 - 2023년 12월
- 역할: 주요 개발자 및 데이터 분석가

## 주요 기술 스택
- 프로그래밍 언어: Python
- 데이터 처리: pandas, numpy
- 자연어 처리: NLTK, spaCy
- 머신러닝: scikit-learn (DBSCAN 클러스터링)
- 딥러닝: Sentence-BERT (텍스트 임베딩)
- 기타: OpenAI API, Anthropic API (LLM 활용)

## 주요 기능 및 구현 방법
1. 숙련 키워드 추출
   - LLM을 활용한 구인공고 텍스트에서 숙련 관련 키워드 추출
   - 정규표현식 기반의 키워드 확장 및 정제

2. 숙련 분류 및 클러스터링
   - LLM을 활용한 초기 숙련 분류
   - DBSCAN 알고리즘을 이용한 유사 숙련 클러스터링

3. 구인공고 자동 매칭
   - 정규표현식 기반의 키워드 매칭
   - 텍스트 유사도 계산을 통한 관련성 높은 숙련 선별

## 주요 성과
- 약 33,000개의 초기 숙련 키워드에서 6,542개의 정제된 숙련 사전 구축
- 500만 건의 구인공고 데이터 처리 및 숙련 매칭 (평균 처리 속도: 8만 건/시간)
- 매칭 정확도 향상: 기존 방식 대비 약 30% 개선 (수동 평가 기준)

## 도전 과제 및 해결 방안
1. 대용량 데이터 처리
   - 해결: 데이터 청크 분할 및 병렬 처리 구현으로 처리 속도 개선
2. 다양한 표현의 숙련 용어 통합
   - 해결: LLM과 클러스터링 기법을 결합하여 유사 숙련 통합 알고리즘 개발

## 학습 및 성장
- 대규모 텍스트 데이터 처리 및 분석 능력 향상
- LLM을 활용한 자연어 처리 기술 습득
- 클러스터링 알고리즘의 실제 적용 경험 획득

## 향후 개선 방향
- 딥러닝 모델을 활용한 숙련 매칭 정확도 추가 개선
- 실시간 구인공고 처리 시스템 구축
- 사용자 피드백을 반영한 지속적인 숙련 사전 업데이트 메커니즘 개발
